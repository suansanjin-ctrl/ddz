# 云上斗地主

一个更接近传统小游戏体验的斗地主项目。

这版不再用手动 WebRTC 配对，而是改成：

- 一个轻量 Python 房间服务器
- 局域网分享链接进房
- `首页 -> 房间页 -> 游戏页` 三段流程
- 服务端统一负责发牌、叫分、出牌校验和同步

## 适合什么场景

- 同一 Wi-Fi 下的几个人一起玩
- 宿舍、办公室、家里这种“周边的人”联机
- 希望像普通网页小游戏一样点链接进房

## 启动方式

在项目目录里运行：

```bash
cd /Users/dzh/Desktop/斗地主
python3 server.py
```

启动后终端会打印两个地址：

- 本机访问地址
- 局域网访问地址

例如：

```text
本机访问: http://127.0.0.1:8000
局域网访问: http://192.168.0.110:8000
```

## 联机流程

1. 房主打开首页，输入昵称，创建房间。
2. 把房间链接发给同一局域网里的另外两个人。
3. 其他人点开链接，输入昵称，直接加入房间。
4. 满 3 人后，房主点击“开始游戏”。
5. 所有人自动跳转到正式牌桌页面。
