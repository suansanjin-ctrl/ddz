# 云上斗地主

一个更接近 QQ 斗地主体验的局域网网页项目。

这版不再用手动 WebRTC 配对，而是改成：

- 一个轻量 Python 房间服务器
- 同一局域网首页大厅自动显示可加入房间
- 首页创建后直接进入牌桌
- 别的设备打开同一个首页地址后可一键入桌
- 等待玩家和正式对局都在同一个牌桌页面里完成
- 服务端统一负责发牌、叫分、出牌校验和同步

## 适合什么场景

- 同一 Wi-Fi 下的几个人一起玩
- 宿舍、办公室、家里这种“周边的人”联机
- 希望像普通网页小游戏一样点链接进房

## 启动方式

在项目目录里运行：

```bash
cd /Users/dzh/Desktop/斗地主
python3 server.py
```

启动后终端会打印两个地址：

- 本机访问地址
- 局域网访问地址

例如：

```text
本机访问: http://127.0.0.1:8000
局域网访问: http://192.168.0.110:8000
```

## 自检方式

如果你想快速确认当前版本能不能正常跑，可以执行：

```bash
cd /Users/dzh/Desktop/斗地主
python3 selftest.py
```

这会自动启动一个临时服务，检查建房、大厅发现、加入、开局、状态同步、非法叫分拦截、重复出牌拦截，以及过期房间清理。

## 联机流程

1. 房主打开首页，输入昵称，创建房间。
2. 创建后会直接进入牌桌等待界面。
3. 其他人打开同一个局域网首页地址。
4. 首页大厅会自动显示当前等待中的牌桌和桌上玩家。
5. 其他人输入昵称后，直接点击“加入这桌”入座。
6. 满 3 人后，房主点击开始，牌桌内直接切换到对局状态。

## 当前页面结构

- `index.html`
  - 首页
  - 创建房间 / 局域网大厅一键入桌

- `game.html`
  - 直达牌桌页面
  - 等待玩家加入
  - 开局后直接进入正式对局
  - 更接近 QQ 斗地主的桌面布局和底部手牌排布

- `room.html`
  - 旧地址兼容跳转

- 叫分流程
