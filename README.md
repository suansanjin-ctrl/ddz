# 云上斗地主

一个更接近 QQ 斗地主体验的网页项目。

当前结构是：

- 一个轻量 Python 房间服务器
- 首页大厅自动显示可加入房间
- 首页创建后直接进入牌桌
- 别人打开同一个地址后可一键入桌
- 等待玩家和正式对局都在同一个牌桌页面里完成
- 服务端统一负责发牌、叫分、出牌校验和同步

## 你真正想要的使用方式

如果你不想每次都在自己电脑上运行终端，那么这个项目应该部署到公网。

部署后就是这种效果：

- 你发一个网址给别人
- 别人直接点开就能进
- 手机可以直接访问
- 不需要你本地先运行 `python3 server.py`

## 推荐运行方式

### 1. 正式使用：部署到支持 Docker 的公网平台

项目已经带了：

- `Dockerfile`
- 自动识别公网域名的分享链接逻辑
- 支持 `PORT` 环境变量

也就是说，任何支持 Docker 的平台都可以直接部署这份仓库。

部署完成后：

1. 打开部署后的公网网址
2. 房主创建房间
3. 把网址发给其他人
4. 大家直接在同一个在线大厅里看到房间并入座

注意：

- 当前房间状态保存在服务进程内存里
- 部署时请保持单实例运行
- 如果服务重启，当前房间会丢失

### 2. 本地调试：继续在电脑上临时运行

如果只是本地开发或自己测试，可以运行：

```bash
cd /Users/dzh/Desktop/斗地主
python3 server.py
```

## 当前联机流程

1. 房主打开首页，输入昵称，创建房间。
2. 创建后会直接进入牌桌等待界面。
3. 其他人打开同一个首页地址。
4. 首页大厅会自动显示当前等待中的牌桌和桌上玩家。
5. 其他人输入昵称后，直接点击“加入这桌”入座。
6. 满 3 人后，房主点击开始，牌桌内直接切换到对局状态。
