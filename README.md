# 云上斗地主

一个更接近 QQ 斗地主体验的局域网网页项目。

这版不再用手动 WebRTC 配对，而是改成：

- 一个轻量 Python 房间服务器
- 局域网分享链接进房
- 首页创建后直接进入牌桌
- 分享链接直接指向牌桌页面
- 等待玩家和正式对局都在同一个牌桌页面里完成
- 服务端统一负责发牌、叫分、出牌校验和同步

## 适合什么场景

- 同一 Wi-Fi 下的几个人一起玩
- 宿舍、办公室、家里这种“周边的人”联机
- 希望像普通网页小游戏一样点链接进房

## 启动方式

在项目目录里运行：

```bash
cd /Users/dzh/Desktop/斗地主
python3 server.py
```

启动后终端会打印两个地址：

- 本机访问地址
- 局域网访问地址

例如：

```text
本机访问: http://127.0.0.1:8000
局域网访问: http://192.168.0.110:8000
```

## 联机流程

1. 房主打开首页，输入昵称，创建房间。
2. 创建后会直接进入牌桌等待界面。
3. 房主把牌桌链接发给同一局域网里的另外两个人。
4. 其他人点开链接后，直接进入同一个牌桌页面，输入昵称入座。
5. 满 3 人后，房主点击开始，牌桌内直接切换到对局状态。
